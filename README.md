Event Connect 🎉

A full-stack college event management platform built using the MERN stack that enables students to discover, register, and manage events seamlessly.

🚀 Features

User authentication with email verification

Create, update, and manage events

Student event registration system

Real-time updates for event activities

Calendar integration to view upcoming events

Role-based access (students & organizers)

🛠️ Tech Stack

Frontend: React.js, HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MongoDB

Other Tools: Nodemailer, REST APIs, GitHub

📌 Project Overview

Event Connect is designed to solve the problem of scattered event information in colleges. It provides a centralized platform where clubs can post events and students can easily browse and register for them.

⚙️ Installation & Setup
1. Clone the repository
git clone [https://github.com/your-username/event-connect.git](https://github.com/Vinaykumarp2005/Event-Connect.git)
cd event-connect
2. Install dependencies
npm install
cd client
npm install
3. Run the project
# Backend
npm run server

# Frontend
cd client
npm start
